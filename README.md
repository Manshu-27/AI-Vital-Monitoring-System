# AI Vital Monitoring System

An AI-powered healthcare analytics system that monitors patient vital signs and predicts blood pressure risk levels using Machine Learning.  
This project is designed as a practical healthcare AI solution suitable for clinics, hospitals, and health-tech startups.

---

## 🚑 Project Overview

The AI Vital Monitoring System processes patient vital data such as blood pressure, heart rate, SpO2, temperature, and respiration rate.  
It automatically:
- Stores patient data
- Generates alerts
- Predicts health risk levels using ML
- Exports data to CSV for analysis and reporting

## 🧠 Machine Learning

- Algorithm: **Random Forest Classifier**
- Features used:
  - Systolic BP
  - Diastolic BP
  - Heart Rate
  - SpO2
  - Temperature
- Output:
  - Risk Level (Normal, Borderline, High, Dangerous)
- Model accuracy is displayed after training.
